# Visão Geral Técnica

O **BFD Diag Helper** é uma aplicação frontend desenvolvida em **React** utilizando **Vite** como bundler.
O projeto foi criado com foco em **organização**, **clareza de código** e **facilidade de evolução**.


## Objetivos técnicos

- Utilizar uma stack moderna de frontend
- Aplicar boas práticas de organização de código
- Facilitar manutenção e escalabilidade
- Preparar o projeto para futura integração com APIs REST

## Escopo atual

- Interface web funcional
- Componentização do frontend
- Estrutura preparada para crescimento
- Sem persistência real de dados
