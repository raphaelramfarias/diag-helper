# Arquitetura do Projeto

O projeto segue uma arquitetura simples e bem definida, baseada na **separação de responsabilidades**.

A ideia principal é manter cada parte da aplicação com uma função clara, facilitando a manutenção
e a evolução do código.

## Visão geral da arquitetura

```text
Usuário
  ↓
Páginas (Pages)
  ↓
Componentes (Components)
  ↓
Serviços / API (futuro)
```
