# Componentes Principais

Este documento descreve os principais componentes da aplicação
e suas responsabilidades.

## Componentes de Interface

### Header
- Responsável pela navegação da aplicação
- Exibe título ou menus principais

### Footer
- Informações complementares
- Créditos ou links úteis

## Componentes Funcionais

### PatientForm
- Responsável pelo cadastro de pacientes
- Controla inputs e validações básicas

### PatientList
- Exibe a lista de pacientes cadastrados
- Organiza informações de forma visual

## Boas práticas adotadas

- Um componente por arquivo
- Componentes pequenos e reutilizáveis
- Nomes claros e descritivos
